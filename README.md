<h2>Для сборки:</h2>
<h3>npm start</h3>
лайв сервер (development mode)
<h3>npm build</h3>
сборка проекта (production mode)
<h3>npm lint / npm lint:fix</h3>
проверка/фикс ESLint(airbnb config)
<h3>npm format</h3>
форматирование prettier
<h3>npm deploy</h3>
деплой на GitHub в ветку 'gh-pages'
